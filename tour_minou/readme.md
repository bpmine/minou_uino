# TOUR DE MINOU

La tour de minou permet à notre chat de trouver une position en hauteur pour mieux observer et défendre son territoire.

Pour le fun, nous avons ajouté des bandeaux de LEDs au-dessus et à l'intérieur.

Un contrôleur NodeMcu de Arduino permet de gérer l'ensemble. Il est accessible par le Wifi de la maison à l'aide d'un webservice basique.
